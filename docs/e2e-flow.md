
End-to-End Flow
1. Bridge Setup
Flash Raspberry Pi OS Lite 64-bit
Run install_bridge.sh
Pair Bluetooth speaker using bluetoothctl
Configure WiFi if needed
Bridge appears as Spotify Connect device
2. Mobile App Installation
Install app from App Store/Google Play or build from source
Open app and discover bridge via mDNS
Enter bridge passphrase to authenticate
Log in to Spotify account
3. Music Playback
Select Campfire Bridge as Spotify output device
Control playback from mobile app:
Play/Pause
Next/Previous track
Volume control
Queue management
Music plays on Bluetooth speaker
Phone audio remains unaffected
4. Advanced Features
iOS
Use AirPlay for app's own audio
Bridge remains primary music player
Android
Samsung devices: Enable Separate App Sound
Non-Samsung: Use Cast or bridge device
Verify setup with test tones
5. Troubleshooting
Check bridge connectivity
Verify Bluetooth connection
Confirm Spotify authentication
Review network settings
